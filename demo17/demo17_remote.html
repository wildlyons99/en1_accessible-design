Demo 17: Random Number Match Project

This project-demo challenges the user to match a random number.

Remote Interface

1. Generate Random Sequence

2. Show Sequence

3. Guess Sequence

4. If you got it right: go back to step #2 to see more of the sequence.

5. Else, if you got it wrong: either try again with this sequence or go back to step #1 to start over with a new sequence.
